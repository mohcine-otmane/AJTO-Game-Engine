AJTO Game Engine Documentation

Welcome to the AJTO Game Engine documentation. This guide will help you install, set up, and use the engine in your projects.

1. Installation

Follow these steps to install the AJTO Game Engine:

  • Download the latest version from the GitHub Repository.
  • Ensure you have SFML installed on your system.
  • Navigate to the downloaded directory and run the following command to compile:
g++ main.cpp -o AJTO -lsfml-graphics -lsfml-window -lsfml-system

2. Getting Started

To create your first game using AJTO, follow these steps:

  • Create a new C++ file and include the engine headers:
#include "Game.hpp"
#include "Actor.hpp"

int main() {
    AJTO game;
    game.SetUp();
    game.GameLoop();
    return 0;
}

3. API Reference

Game Class

The Game class is the base class for all games in AJTO:

  • void SetUp() - Initializes the game.
  • void GameLoop() - Starts the game loop.
  • void UpDate(float deltaTime, sf::Event event) - Updates the game state.

#include "Actor.hpp"
class MyGame : public Game {
public:
    Actor* player;

    void SetUp() override {
        player = new Actor(sf::Vector2f(100.f, 100.f), sf::Vector2f(10.f, 10.f));
        player->AddAnimation("Walk", sf::Vector2i(8, 1), true);
        Actors["player"] = player;
    }

    void UpDate(float deltaTime, sf::Event event) override {
        if (signalSystem.HasEmitted("MoveRight")) {
            player->position.x += player->speed.x * deltaTime;
        }
    }
};

int main() {
    MyGame game;
    game.SetUp();
    game.GameLoop();
    return 0;
}
